# SKILL: Generate Custom Magic Card

## Description
A personal skill to generate a new, custom, and unique Magic: The Gathering card using Scryfall API fetching for inspiration. This skill guides you through a step-by-step workflow to create a card, ensuring creativity and uniqueness.

## Workflow
1. **Fetch Inspiration**
   - Query Scryfall API for random or filtered card data.
   - Extract interesting mechanics, flavor, or art cues.
2. **Ideate Custom Card**
   - Brainstorm a new card concept based on Scryfall inspiration.
   - Decide card type, color, rarity, and theme.
3. **Draft Card Details**
   - Write card name, mana cost, type line, rules text, flavor text, and stats.
   - Ensure uniqueness and balance.
4. **Format Output**
   - Save card as Markdown, JSON, or your preferred format.
   - Optionally include Scryfall reference or inspiration notes.
5. **Completion Check**
   - Confirm card is original, inspired, and formatted correctly.
   - Optionally review for balance and creativity.

## Decision Points
- Choose random or filtered Scryfall inspiration.
- Select output format (Markdown, JSON, etc.).
- Decide how much to borrow from Scryfall vs. invent.

## Quality Criteria
- Card is unique and not a direct copy.
- Inspiration is clearly referenced.
- Output is well-formatted and ready for use.

## Example Prompts
- "Generate a new custom Magic card using Scryfall inspiration."
- "Create a unique blue creature card, inspired by Scryfall."
- "Draft a legendary artifact based on random Scryfall card."

## Related Customizations
