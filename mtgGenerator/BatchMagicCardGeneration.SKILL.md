# SKILL: Batch Magic Card Generation

## Description
A personal skill to generate multiple custom Magic: The Gathering cards in one session, using Scryfall API fetching for inspiration.

## Workflow

1. Define batch size and card parameters (color, type, rarity, etc.)
2. For each card:
   - Fetch Scryfall inspiration
   - Ideate and draft card details
   - Format and save output
3. Review batch for uniqueness and balance

## Example Prompts
- "Generate 10 custom Magic cards, inspired by Scryfall."
- "Create a batch of red creatures."
