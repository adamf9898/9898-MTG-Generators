# SKILL: Custom Card Web Generator (HTML/CSS/JS)

## Description
A personal skill to create a web-based generator for custom Magic cards, using Scryfall API fetching for inspiration.

## Workflow

1. Design HTML/CSS layout for card display
2. Implement JavaScript to fetch Scryfall data
3. Generate and display custom card details
4. Allow user input for card customization
5. Export or share generated cards

## Example Prompts
- "Build a web app for custom Magic card generation."
- "Create HTML/CSS/JS for a card generator using Scryfall."
