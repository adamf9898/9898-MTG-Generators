# SKILL: MTG Card Text/Image Integration

## Description
A personal skill to integrate custom Magic card text and images, using Scryfall API and generation tools.

## Workflow

1. Fetch Scryfall card image or art
2. Generate custom card text
3. Combine text and image in preferred format (Markdown, HTML, etc.)
4. Save or export for sharing

## Example Prompts
- "Create a custom card with Scryfall art."
- "Integrate card text and image for a new card."
